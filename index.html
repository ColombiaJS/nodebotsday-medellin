NodeBots Day 7/27 - Medellín

El próximo 27 de Julio del 2013 se celebrará el NodeBots Day, un evento internacional donde estaremos realizando Hardware Hacking con Node.js.

En Medellín se realizará dentro de la 5ta edición del BarCamp Med - tejido social digital!

Requisitos para participar

  • Tener instalada la última version estable de Node.js (v0.10.x), la pueden descargar de nodejs.org/download
  • Instalar el paquete nodebotsdaymed13 desde npm, ejecutando npm install nodebotsdaymed13

Bring Your Own Hardware!

Si tienes alguno de los siguientes dispositivos, tráelos para que sea mucho mas divertido!

  • Raspberry Pi
  • Arduino Uno
  • Parrot AR Drone 2.0
